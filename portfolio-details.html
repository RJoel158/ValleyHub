Chapa inteligente con reconocimiento facial

Este proyecto explora las distintas posibilades de trabajo con herramientas electronicas y software con aplicacion de un modelo de aprendizaje supervisado.

Para el circuito electronico se hizo uso de los siguientes items:
~Arduino Uno SMD
~Modulo Relay
~Chapa Solenoide 12v
~Resistencias 220Ω
~Jumpers

Proyecto pensado en la efectividad de los actuales sistemas de seguridad con una verificacion de dos pasos de manera intuitiva.

Joel Saavedra && Rodrigo Gutierrez

Código de reconocimiento facial


import cv2
import numpy as np

# Carga del modelo entrenado
model = cv2.face.LBPHFaceRecognizer_create()
model.read("modelo_entrenado.xml")

# Inicialización de la cámara
camera = cv2.VideoCapture(0)

while True:
    ret, frame = camera.read()
    if not ret:
        break

    # Convertir a escala de grises

    for (x, y, w, h) in faces:
        roi_gray = gray[y:y + h, x:x + w]
        id, confidence = model.predict(roi_gray)
        if confidence < 50:
            cv2.putText(frame, f"ID: {id}", (x, y - 10), cv2.FONT_HERSHEY_SIMPLEX, 0.9, (0, 255, 0), 2)
        else:
            cv2.putText(frame, "Desconocido", (x, y - 10), cv2.FONT_HERSHEY_SIMPLEX, 0.9, (0, 0, 255), 2)

    cv2.imshow("Reconocimiento Facial", frame)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

camera.release()
